For installing the MINDED-FBA plugins, go through the following steps:
1 - Open the QGIS software.
2 - In QGIS, select the option: "Settings> User profiles> Open active profile folder"
3 - Extract and copy the downloaded "minded_fba" folder to "/python/plugins/", which is located within the active profile folder.
4 -  Restart the QGIS software and select the option: "Plugins>Manage and Install plugins".
5 - Search for "MINDED FBA" and install the plugin.
6 - Start the plugin from the QGIS plugins toolbar.
